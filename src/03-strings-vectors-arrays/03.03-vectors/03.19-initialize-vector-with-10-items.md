1. `vector<int> v1(10, 42);`
2. `vector<int> v2 = {42, 42, 42, 42, 42, 42, 42, 42, 42, 42};`
3. ```cpp
vector<int> v3;
for (unsigned int i = 0; i < 10; ++i) {
    v3.push_back(42);
}
```

Preferred way is 1, as it initializes elements directly without an extra copy (2.) or even possibly resizing (3.)
